🛒 Lista de Compras
Uma aplicação web simples, intuitiva e responsiva para gerenciamento de itens de compra. Ideal para organizar o que você precisa comprar antes de ir ao mercado, permitindo adicionar, marcar como concluído e excluir itens.

## 🚀 Tecnologias

📋 Funcionalidades
Adicionar Itens: Insira novos produtos na lista rapidamente.

Checklist: Marque itens já coletados com um clique.

Remoção: Exclua itens individualmente ou limpe a lista.

Responsividade: Design adaptado para dispositivos móveis e desktop.

Feedback Visual: Notificações ou estados visuais ao interagir com a lista.
