# 🛒 Lista de Compras

Uma aplicação web simples, intuitiva e responsiva para o gerenciamento de itens de compra. Ideal para organizar suas necessidades antes de ir ao mercado, permitindo adicionar, marcar como concluído e excluir itens com facilidade.

## 📋 Funcionalidades

* **Adicionar Itens:** Insira novos produtos na lista rapidamente.
* **Checklist:** Marque itens já coletados com um clique.
* **Remoção:** Exclua itens individualmente para manter a lista limpa.
* **Responsividade:** Design adaptado para dispositivos móveis e desktop.
* **Interface Limpa:** Foco total na experiência do usuário e usabilidade.

## 💻 Como Rodar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/deivid94/lista_de_compras.git](https://github.com/deivid94/lista_de_compras.git)
    ```
2.  **Acesse a pasta:**
    ```bash
    cd lista_de_compras
    ```
3.  **Execução:**
    Abra o arquivo `index.html` diretamente no seu navegador ou utilize a extensão **Live Server** no VS Code para uma melhor experiência de desenvolvimento.
